¿Como se calcula?

Cuando el niño tiene hasta 30kg: Se utiliza el calculo de Holliday-Segar.

  1. Cuando el niño tiene hasta 10kg: 100cc por cada kilo
  2. Luego, por cada kilo arriba de 10 y hasta 20kg, se suman 50cc
  3. Finalmente, por cada kilo arriba de 20kg se suman 20cc
  4. Para calcular el mantenimiento, dividimos el resultado en 24 para conocer los cc necesarios por hora.
  5. El medio mantenimiento se obtiene multiplicando el mantenimiento x 1.5

Cuando el niño pesa mas de 30kg, se utiliza el Método de superficie corporal:

  1. Superficie corporal = ( (peso * 4) + 7) / (peso + 90)
  2. Este resultado se multiplica por 1500 o por 2000 para hallar el valor del volumen diario en cc, y el medico decide cual de los dos resultados utilizar.
